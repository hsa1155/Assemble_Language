在我的程式中 首先我先呼叫malloc來為陣列配置空間
傳入整數r0 代表空間大小 回傳r0代表陣列位址

接下來呼叫rand函數為陣列賦值 無傳入 return r0=一隨機整數
用迴圈的方法一個一個填入一個隨機的值

接下來呼叫自己寫的numsort（實做方法為bubble sort）
傳入r0 r1 代表陣列大小以及位址
回傳r0 一個全新排序好的陣列

之後參考給的範例程式
進行open write close
open與範例方法相同

接下來為了寫入一個字串
先呼叫了malloc函式產生空間來存放最後的結果（line 92）

接下來 對於陣列中每一個數字
呼叫sprintf函式把它轉為自串
傳入r0,r1,r2    r0為轉為字串後位址 r1為字串格式 r2為目標數字
不會傳回新的字串位址 而是直接改動傳入的r0

之後呼叫strcat函式將其接在最後結果字串後面
傳入r0 r1   r0為前面的字串 r1為接在後面的字串

在迴圈結束後
呼叫strlen函式來確認總長度
傳入r0 代表字串位址 回傳r0 代表長度
接下來的write與close步驟與範例程式相同

編譯時使用makefile
執行時使用insight來執行
